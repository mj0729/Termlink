# 模块索引

- `app-shell`: 应用壳层、入口注册、全局消息与主题
- `ui-components`: Vue 组件层，含菜单、标签页、侧边栏、模态框、文件管理、监控与下载面板
- `services`: SSH、SFTP、主题等前端服务
- `desktop-backend`: Tauri Rust 侧能力，当前迁移仅做兼容性回归关注，不计划改动
